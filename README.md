# CareerGuide AI

An intelligent career assistant that helps students discover their ideal career path through AI-powered analysis. Users complete a simple profile (interests, skills, subjects, goals), and the system generates:

- ✅ Top 3 personalized career matches with match percentages
- 🔍 Skill gap analysis (strengths vs missing skills)
- 🗺️ 6-month structured learning roadmap
- 🛠️ Portfolio project ideas tailored to target career
- 📄 Professional cover letter drafts
- 🎤 Interview preparation (technical + behavioral Q&As)
- 💬 AI Mentor chat for follow-up questions

**Built for SDG 4 - Quality Education:** Bridging the career guidance gap for students in rural and semi-urban areas with no access to counselors.

**Tech Stack:** Streamlit | Python | Groq API (Llama 3.3 70B) | Hugging Face Spaces
